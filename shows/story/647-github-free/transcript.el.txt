που καλύπτει τα κορυφαία εγχειρίδια της εβδομάδας όπως το Linux bias github που ανακοινώθηκε την περασμένη εβδομάδα
ότι όλα τα βασικά χαρακτηριστικά του είναι τώρα διαθέσιμα δωρεάν σε όλους τους χρήστες που
σημαίνει απεριόριστα ιδιωτικά αποθετήρια με απεριόριστες συνεργασίες για όλους
συμπεριλαμβανομένων ομάδων που χρησιμοποιούν την υπηρεσία για εμπορικά έργα καθώς και έως
2.000 λεπτά το μήνα δωρεάν πρόσβαση σε ενέργειες github αυτοματισμού της εταιρείας
και CI CD πλατφόρμες ομάδες που θέλουν πιο προηγμένες δυνατότητες όπως κατόχους κώδικα ή
εταιρικές λειτουργίες όπως η υποστήριξη saml θα πρέπει ακόμη να αναβαθμιστούν σε πρόγραμμα πληρωμής
αυτή τη στιγμή, αλλά η τιμή για αυτά τα σχέδια έχει μειωθεί στο μισό
Η εταιρεία ακολουθούσε πάντα μια προσέγγιση freemium στο μοντέλο τιμολόγησης αλλά από τότε
την απόκτησή της από τη Microsoft άρχισε να επεκτείνει τον αριθμό των χαρακτηριστικών του
Ο Διευθύνων Σύμβουλος των δωρεάν λογαριασμών github, Matt Friedman, τόνισε ότι αυτή η κίνηση ήταν από καιρό
ο χάρτης πορείας και δεν είναι μια περιορισμένη προώθηση που βασίζεται στο τρέχον Kovan
19 κρίση λέει ότι αυτό είναι κάτι που σκοπεύουμε να κάνουμε και θέλαμε να κάνουμε για ένα
πολύ καιρό αφού ουσιαστικά κάναμε την απόκτηση και τώρα φτάνουμε σε αυτό
για να το κάνω χρειάστηκε μέχρι τώρα μέχρι να γίνει υψηλή προτεραιότητα
