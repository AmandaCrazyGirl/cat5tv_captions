өткөн жумада жарыяланган Linux bias github сыяктуу жуманын эң мыкты окуу куралдары
анын бардык негизги өзгөчөлүктөрү эми бардык колдонуучуларга бекер берилет
баардыгы үчүн чексиз кызматташуу менен чексиз жеке репозиторийлерди билдирет
анын ичинде коммерциялык долбоорлор үчүн кызматты колдонгон топтор
Айына 2000 мүнөт компаниянын автоматташтыруу иш-аракеттерине github акысыз кирүү мүмкүнчүлүгүн берет
жана CI CD платформасынын командалары код ээлери же дагы башка өнүккөн функцияларды талап кылышат
saml колдоо сыяктуу ишкананын өзгөчөлүктөрү дагы эле төлөө планына чейин көтөрүлүшү керек
ушул учурда, бирок ошол пландар үчүн баа жарымында кыскарган
компания өзүнүн баа моделине ар дайым фремиумдук мамиле жасайт, бирок андан бери
аны Майкрософт тарабынан сатып алганда, анда анын функциялары кеңейе баштады
акысыз эсептер github компаниясынын директору Мэтт Фридман бул кадам узак убакытка созулганын баса белгиледи
жол картасы жана ал учурдагы Кован тарабынан демилгеленген чектелген жарнамалоо эмес
19 кризис, ал бул биз жасай турган жана биз үчүн жасагысы келген нерсе
көптөн бери биз сатып алганбыз жана азыр буга жетиштик
бул эң маанилүү приоритет болуп калганга чейин ушул убакка чейин жасалды
