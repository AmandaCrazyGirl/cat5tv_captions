фарогирии китобҳои дарсии ҳафта ба монанди Linux bias github дар ҳафтаи гузашта эълом шуд
ки ҳоло ҳамаи хусусиятҳои асосии он барои ҳамаи корбарон ройгон дастрасанд
маънои анборҳои шахсии номаҳдуд бо ҳамкории бемаҳдуд барои ҳама
аз он ҷумла дастаҳое, ки хидматро барои лоиҳаҳои тиҷоратӣ ва то охир истифода мебаранд
2,000 дақиқа дастрасии ройгон ба автоматикаи ширкат ба github
ва дастаҳои платформаи CD CI, ки хусусиятҳои бештар пешрафтаро ба монанди соҳибони рамзҳо ё мехоҳанд
хусусиятҳои корхона ба монанди дастгирии saml ҳанӯз ҳам бояд ба нақшаи пардохт такмил дода шаванд
дар айни замон, аммо нархгузорӣ барои ин нақшаҳо дар нисф кам карда шудааст
ширкат ҳамеша ба намунаи нархгузории худ муносибати озодона истифода мекунад, аммо аз он вақт инҷониб
Пас аз он, ки Microsoft онро харидорӣ кардааст, ба васеъ кардани шумораи хусусиятҳои он шуруъ кардааст
аккаунтҳои озод github директори Мэтт Фридман қайд кард, ки ин иқдом дер давом карда буд
