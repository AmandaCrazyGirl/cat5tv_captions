ครอบคลุมตำราเรียนยอดนิยมประจำสัปดาห์เช่น Linux bias github ประกาศเมื่อสัปดาห์ที่แล้ว
คุณลักษณะหลักทั้งหมดของมันพร้อมให้ผู้ใช้ทุกคนสามารถใช้ได้ฟรี
หมายถึงที่เก็บส่วนตัวไม่ จำกัด พร้อมความร่วมมือที่ไม่ จำกัด สำหรับทุกคน
รวมถึงทีมที่ใช้บริการสำหรับโครงการเชิงพาณิชย์รวมถึงสูงสุด
2,000 นาทีต่อเดือนฟรีการเข้าถึง GitHub การกระทำอัตโนมัติของ บริษัท
และทีมแพลตฟอร์มซีดี CI ที่ต้องการคุณสมบัติขั้นสูงเพิ่มเติมเช่นเจ้าของรหัสหรือ
คุณสมบัติขององค์กรเช่นการสนับสนุน saml จะยังคงต้องอัพเกรดเป็นแผนจ่ายเงิน
ในขณะนี้ แต่การกำหนดราคาสำหรับแผนเหล่านั้นถูกลดลงครึ่งหนึ่งแล้ว
บริษัท ได้ใช้วิธีการกำหนดราคาแบบ freemium อยู่ตลอดเวลา แต่เนื่องจาก
การเข้าซื้อกิจการโดย Microsoft มันเริ่มขยายจำนวนคุณลักษณะใน
แมตต์ฟรีดแมนซีอีโอของ Github กล่าวว่าบัญชีฟรีโดยเน้นว่าการดำเนินการนี้ดำเนินมายาวนาน
แผนงานและมันไม่ได้เป็นโปรโมชั่น จำกัด แรงบันดาลใจจาก Kovan ปัจจุบัน
เขาพูดว่านี่เป็นสิ่งที่เราวางแผนจะทำและต้องการทำเพื่อ
นานมาแล้วที่เราทำการซื้อและตอนนี้มาถึงนี้
