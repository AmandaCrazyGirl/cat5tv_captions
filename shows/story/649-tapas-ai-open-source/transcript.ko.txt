리눅스 바이어스 구글과 같은 주 최고의 교과서를 다루는 구글은 코드를 발표했다
내부적으로 개발 된 인공 지능 타파스를 통해
최신 iPhone의 이름과 같은 자연어 질문을하고
관계형 데이터베이스 또는 스프레드 시트에서 답변을 가져 오십시오.
검색 대기업의 연구원들이 목요일에 AI를 자세히 공개 한 오픈 소스
타파스는 Bert를 기반으로하는 자연어 처리 기술을 기반으로합니다.
Google은 검색 엔진에서 전 세계의 상당 부분을 사용합니다.
정보는 행과 열로 구성된 관계형입니다.
역사적으로 수동으로 필요한 행과 열에서 탐색
스프레드 시트를 통해 리프팅 또는 SQL 쿼리 자연어 작성
처리하면 사용자가 작업을 훨씬 쉽게 수행 할 수 있습니다.
이 기술은 Google 및 기타 업체에서 광범위하게 채택되었습니다.
분석 시장 검색 거인은 가장 힘든 비트 또는 일치하는
관계형 데이터를 분석하기위한 세 가지 최고의 오픈 소스 알고리즘
영어 버전의 Wikipedia에서 620 만 개의 테이블에 AI
3 가지의 학술 데이터 세트 벤치 마크 테스트에서 작동하도록 설정
신경망이 라이벌과 비교할 수있는 정확한 답변을 제공한다는
세 가지 데이터에 걸친 알고리즘은 Google의 언어 처리 유형을 설정합니다.
타파스로 구현하여 AI가 제기 된 질문뿐만 아니라
쿼리하려는 데이터의 사용자뿐만 아니라 관계형의 구조
데이터가 저장된 타파스 테이블은 데이터 가져 오기를 넘어
예를 들어 비즈니스 사용자가 평가하는 경우 기본 계산 수행
판매 데이터는 회사의 세 곳에서 평균 수익을 요구했습니다.
AI가 데이터뿐만 아니라 계산 된 답변으로 응답하는 인기 제품
