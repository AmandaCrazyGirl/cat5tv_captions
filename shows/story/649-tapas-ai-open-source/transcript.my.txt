ယခုအပတ်တွင် Linux ဘက်လိုက်သောဂူဂဲလ်ကဲ့သို့သောထိပ်တန်းဖတ်စာအုပ်များအကြောင်းရေးသားထားသောဂူဂဲလ်သည် code ကိုထုတ်ပြန်ခဲ့သည်
သူတို့ရဲ့ပြည်တွင်းရွှေ့ပြောင်းတီထွင်ထားသောအတုထောက်လှမ်းရေးတာပက်များအတွက်သူတို့လုပ်နိုင်သည်
နောက်ဆုံးပေါ်အိုင်ဖုန်းနှင့်ဘာကဲ့သို့သောသဘာဝဘာသာစကားမေးခွန်းတစ်ခုကိုယူပါ
အဖြေကို relational database သို့မဟုတ် spreadsheet မှရယူပြီးယခုဖြစ်သည်
open source ရှာဖွေရေးကုမ္ပဏီကြီးရဲ့သုတေသီများကြာသပတေးနေ့တွင် AI အသေးစိတ်
tapas သည် Bert ကိုသဘာဝဘာသာစကားပြုပြင်သည့်နည်းစနစ်ကိုအခြေခံသည်
ဂူးဂဲလ်သည်၎င်း၏ရှာဖွေရေးအင်ဂျင်တွင်ကမ္ဘာ့အကြီးအကျယ်နေရာတစ်ခုကိုအသုံးပြုသည်
သတင်းအချက်အလက် relational ဖြစ်ပါတယ်အတန်းနှင့်ကော်လံသို့စည်းရုံးပြောဆိုလိုသည်
ဤအအတန်းများနှင့်ကော်လံများမှသမိုင်းကြောင်းအရဖြစ်စေလက်ဖြင့်လိုအပ်သည်
shift spreadsheet မှတဆင့် lifting သို့မဟုတ် SQL query ကိုသဘာဝဘာသာစကား
အပြောင်းအလဲနဲ့လုပ်ခြင်းသည်အသုံးပြုသူများအတွက်လုပ်ငန်းတာဝန်ကိုသိသိသာသာပိုမိုလွယ်ကူစေသည်
နည်းပညာကို Google နှင့်အခြားကစားသမားများကကျယ်ကျယ်ပြန့်ပြန့်အသုံးပြုခဲ့သည်
ခွဲခြမ်းစိတ်ဖြာမှုခွဲခြမ်းစိတ်ဖြာမှုစျေးကွက်ရှာဖွေရေးကုမ္ပဏီကြီးသည်အပြင်းထန်ဆုံးအနိုင်ယူသို့မဟုတ်ကိုက်ညီကပြောပါတယ်
သူတို့လေ့ကျင့်ပေးသော relational data များကိုခွဲခြမ်းစိတ်ဖြာခြင်းအတွက်ထိပ်တန်း open-source algorithms သုံးခု
အင်္ဂလိပ်ဝီကီပီးဒီးယားမှဇယား ၆.၂ သန်းနှင့် AI
ပြသထားသည့်ပညာရေးဆိုင်ရာအချက်အလက်များအခြေခံစံနှုန်းစမ်းသပ်မှုသုံးခုကိုလုပ်ဆောင်ရန်သတ်မှတ်ထားသည်
အာရုံကြောကွန်ယက်သည်ပြိုင်ဘက်အဖြစ်တိကျမှန်ကန်သောနှိုင်းယှဉ်နိုင်သောအဖြေများကိုပေးသည်ဟုဆိုသည်
ဒေတာသုံးခုစလုံးတွင်ပြုလုပ်ထားသော algorithms သည်ဂူးဂဲလ်၏ဘာသာစကားအပြောင်းအလဲဖြစ်သည်
tapas သို့အကောင်အထည်ဖော်လျက်ရှိသည် AI ကိုမေးခွန်းထုတ်မသာစဉ်းစားရန်ခွင့်ပြုပါတယ်
အသုံးပြုသူများကသူတို့မေးမြန်းချင်ပေမယ့် relational ၏ဖွဲ့စည်းပုံမှာ
ဒေတာများကိုသိမ်းဆည်းထားသည့်ဇယားများ tapas သည်အချက်အလက်ရယူခြင်းထက်ကျော်လွန်နိုင်သည်
စီးပွားရေးလုပ်ငန်းအသုံးပြုသူတစ် ဦး အကဲဖြတ်လျှင်လည်းဥပမာအခြေခံတွက်ချက်မှုလုပ်ဆောင်ပါ
ရောင်းအားဒေတာကသူတို့ကုမ္ပဏီသုံးခုရဲ့ပျမ်းမျှဝင်ငွေကိုမေးတယ်
လူကြိုက်များသောထုတ်ကုန်များက AI သည်အချက်အလက်များကိုသာမကတွက်ချက်သည့်အဖြေဖြင့်ပြန်လည်ဖြေကြားလိမ့်မည်
