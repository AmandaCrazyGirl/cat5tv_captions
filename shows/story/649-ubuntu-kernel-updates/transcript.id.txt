beberapa kerentanan ditemukan belakangan ini oleh berbagai peneliti keamanan
mempengaruhi Linux empat koma satu empat empat koma dan lima koma tiga kernel
dan beberapa versi Ubuntu Linux, patch keamanan baru memperbaiki masalah yang ditemukan
pada driver Intel Wi-Fi kondisi ras ditemukan di kernel Linux
implementasi terminal virtual ditemukan cacat pada driver floppy dan
kondisi ras di blok IO melacak implementasi semua masalah ini bisa
memungkinkan penyerang lokal untuk crash sistem atau mengekspos sensitif
informasi pembaruan kernel baru juga menambal tumpukan buffer overflow
ditemukan di driver v host net ini dapat memungkinkan penyerang lokal dengan
kemampuan untuk melakukan panggilan ioctl di dev v host net untuk menyebabkan penolakan layanan
menabrak sistem yang hanya untuk menyebutkan beberapa masalah keamanan penting
kanon yang telah ditambal mendesak semua pengguna untuk memperbarui instalasi mereka
dan instal versi kernel baru secepat mungkin
