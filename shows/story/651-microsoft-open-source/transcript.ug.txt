بۇ ھەپتىدىكى ئەڭ يۇقىرى تېخنىكىلىق ھېكايىلەرنى 2011-يىلى ئازراق linux بىر تەرەپلىمىلىك بىلەن قاپلىغان
2001-يىلدىكى مىكروسوفت ئەينى ۋاقىتتىكى ستېۋ توپچى
داڭلىق ماركىلىق linux ئۆزىگە چاپلانغان راك
ئۇ تۇتقان ھەممە نەرسىگە بىلىم مۈلۈك ھوقۇقى مەنىسىدە
ئەمما microsoft ئوچۇق كود ھەققىدە خاتا ئىكەنلىكىنى ئېتىراپ قىلدى
شىركەت ئۈستەل يۈزىنىڭ ئېگىزلىكىدە نەچچە يىل جەڭ قىلغاندىن كېيىن
ھازىر چوشقىلار ئۇچۇۋاتىدۇ
microsoft نىڭ ھازىرقى پرېزىدېنتى براد سىمىس شىركەتنىڭ خاتالىقىغا ئىشىنىدۇ
ئوچۇق كودتا ئۇ مىكروسوفتنىڭ خاتا تەرەپتە ئىكەنلىكىنى ئېيتتى
تارىخنىڭ بېشىدا ئوچۇق كود پارتىلىغاندا تارىخ
ئەسىر ۋە مەن ئۆزۈم ھەققىدە شۇنداق دېيەلەيمەن
smith نىڭ مىكروسوفتتا تۇرۇۋاتقىنىغا 25 يىلدىن ئاشتى
ھەمدە ئوچۇق-ئاشكارە جەڭ قىلىش جەريانىدا شىركەتنىڭ ئالىي ئادۋوكاتلىرىنىڭ بىرى
مەنبە يۇمشاق دېتالى ئۇ خۇش خەۋەرنى قوشۇپ قويدى ، ئەگەر ھايات بولسا
ئۇزۇن ۋاقىت ئىچىدە مىكروسوفتنىڭ ئۆزگەرتىشى كېرەكلىكىنى ئۆگىنەلەيسىز
ئەلۋەتتە linux راك كېسىلىگە گىرىپتار بولغان كۈندىن باشلاپ ئۆزگەردى
يۇمشاق دېتال ماگناتى ھازىر ئوچۇق كودقا ئەڭ كۆپ تۆھپە قوشقان
دۇنيادىكى تۈرلەر facebook docker google apache نى مەغلۇب قىلدى
يەنە نۇرغۇن كىشىلەر مىكروسوفت ئاستا-ئاستا بارلىققا كەلدى
يېقىنقى يىللاردا ئوچۇق مەنبەلىك توك مەنبەسىنى ئۆز ئىچىگە ئالغان ئوچۇق كودنى قوللىنىش
كۆرۈنمە چاستوتا كودى ھەتتا microsoft edger نىڭ ئەسلى javascript ماتورى
microsoft يەنە Canonical بىلەن ھەمكارلىق ئورنىتىپ ، ئۇبۇنتۇنى Windows 10 غا ئېلىپ كەلدى
ھەمدە كۆچمە ئەپ ئېچىش ۋە github نى ئاسراشقا ياردەم بېرىش ئۈچۈن xamarin نى سېتىۋالغان
ئاچقۇچىلار مىكروسوفتنىڭ مودا كود ئامبىرى
ھەتتا Windows 10 يېڭىلاشتا تولۇق linux مېغىزى ئەۋەتىلىدۇ
ئۇ مۇشۇ ئاينىڭ ئاخىرىدا تارقىتىلىدۇ ۋە ئۇ خىروم توركۆرگۈ ماتورىغا يۆتكەلدى
ئالدىنقى يىلى microsoft مۇ ھەمكارلىشىۋاتىدۇ
ئوچۇق كودلۇق مەھەللە ئويۇنچۇقلارنى بارلىققا كەلتۈرىدۇ
Windows 10 ۋە شىركەتنىڭ ھازىر ئوچۇق لايىھىلەش پەلسەپىسى ئۈچۈن
پەلسەپە بەلكىم بىز Windows دا تېخىمۇ كۆپ ئوچۇق كود تىرىشچانلىقىنى كۆرىدىغانلىقىمىزدىن دېرەك بېرىدۇ
