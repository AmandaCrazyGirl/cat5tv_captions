၎င်းတို့သည် Ubuntu ကိုကြိုတင်တင်ထားသည့် OS option တစ်ခုသို့မဟုတ်နံပါတ်များတွင်ထည့်သွင်းထားသည်
သူတို့ရဲ့နွေရာသီ Lenovo မှစပြီးသူတို့ရဲ့စနစ်များကိုကောင်းစွာကိုယ်စားပြုထားသည်
linux ဟာ့ဒ်ဝဲအသိုင်းအဝိုင်းအတွင်းက၎င်း၏ထုတ်ကုန်တစ်ခု swath အသိအမှတ်ပြုထားပြီးမှ
အမျိုးမျိုးသောကွဲပြားခြားနားသော distros များအတွက်နှစ်ပေါင်းများစွာနှင့်ကုမ္ပဏီမကြာသေးမီကထုတ်ဖော်ပြောကြားခဲ့သည်
Fedora နှင့်ကြိုတင်ထည့်သွင်းထားသည့်လက်ပ်တော့ပ်များကိုရောင်းချရန်နှင့် firmware အသစ်များပိုမိုပြုလုပ်ရန်စီစဉ်နေသည်
အဆိုပါရောင်းချသူကြားနေ linux ရောင်းချသူ firmware ကိုဝန်ဆောင်မှုမှတဆင့်ရရှိနိုင်ပေမယ့်ယခု
Lenovo က Linux ချစ်ခြင်းမေတ္တာနဲ့အတူဆက်ပြီးလုပ်ဆောင်သွားမှာပါ

LTS သည်ကြို။ တင်ထားပြီးသတ်သတ်မှတ်မှတ်စီစဉ်ထားမှုအနည်းငယ်သာရှိသည်
တူညီသောထုတ်ကုန်များကိုဝယ်ယူနိုင်စဉ်တစ်နေရာရာတွင်သိုလှောင်မှုစာမျက်နှာကိုရှာဖွေရန်ခက်ခဲသည်
Windows 10 သို့မဟုတ် Red Hat Enterprise Linux ကိုကြိုတင် install လုပ်ပြီးပါက၎င်းသည်အဓိကဖြစ်နေဆဲဖြစ်သည်
Ubuntu နှင့် Linux အသိုင်းအဝိုင်းအတွက်အနိုင်ရပါကကုမ္ပဏီကပြောကြားလိမ့်မည်
တည်ငြိမ်မှုကိုထိန်းသိမ်းရန်ကူညီနိုင်ရန်အတွက် Linux kernel သို့ upstream device drivers များကိုတိုက်ရိုက်ဖြစ်သည်
နှင့် workstation ၏ဘဝတစ်လျှောက်လုံးလိုက်ဖက်တဲ့သည်လွယ်ကူသည်
၀ င်းဒိုးကို သုံး၍ စဉ်းစားထားသောစဉ်းစားတွေးခေါ်မှုများဝယ်ယူရန်နှင့် Ubuntu ကိုကိုယ်တိုင်ထည့်သွင်းရန်လုံလောက်သည်
တရားဝင်ပံ့ပိုးမှုဆိုသည်မှာ ၀ ယ်ယူသူများသည်ဟာ့ဒ်ဝဲအားလုံးနှင့်အလုပ်လုပ်ကြောင်းသေချာစေသည်
Ubuntu box ထဲမှာရှာဖွေလို့မရတဲ့တည်းဖြတ်ဖို့အတွက် file တွေကိုလိုက်ရှာဖို့ (သို့) config file လုပ်ဖို့ driver မရှိဘူး
၀ င်းဒိုးလိုင်စင်မလိုအပ်ဘဲကုန်ကျစရိတ်သက်သာစေသည်
Ubuntu LTS ၏သက်တမ်းသည်ငါးနှစ်တိုးမြှင့်ခြင်း၏အကျိုးကျေးဇူးဖြစ်သည်ဟု Lenovo ကပြောကြားခဲ့သည်
