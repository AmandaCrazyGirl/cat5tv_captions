ครอบคลุมตำราเรียนชั้นนำของสัปดาห์อย่าง Linux bias lenovo ได้ประกาศว่า
พวกเขารวมถึง Ubuntu เป็นตัวเลือกระบบปฏิบัติการที่โหลดไว้ล่วงหน้าหรือตัวเลขในจำนวน
ระบบของพวกเขาเริ่มต้นในช่วงฤดูร้อนนี้เลโนโวมีตัวแทนอยู่แล้ว
ภายในชุมชนฮาร์ดแวร์ linux ที่ได้รับการรับรองอุปกรณ์ของตน
สำหรับ distros ที่แตกต่างหลากหลายในช่วงหลายปีที่ผ่านมาและ บริษัท เพิ่งเปิดเผย
วางแผนขายแล็ปท็อปที่โหลดไว้ล่วงหน้ากับ fedora และทำการอัปเดตเฟิร์มแวร์เพิ่มเติม
พร้อมใช้งานผ่านทางบริการเฟิร์มแวร์ของผู้จำหน่าย linux ที่เป็นกลาง แต่ตอนนี้
มันจะพัฒนาต่อไปด้วยเลโนโวเลโนโวเลิฟกล่าวว่าทุกอย่างคิด
สถานีและแล็ปท็อป ThinkPad ซีรี่ส์ซีรีส์จะสามารถซื้อได้ด้วยบาดแผล -
LTS ถูกโหลดไว้ล่วงหน้าและไม่ใช่เฉพาะการกำหนดค่าบางอย่างที่เก็บไว้
เป็นเรื่องยากที่จะหาหน้าจัดเก็บข้อมูลที่ไหนสักแห่งในขณะที่สามารถซื้ออุปกรณ์เดียวกันได้
ด้วย Windows 10 หรือ Red Hat Enterprise Linux ที่ติดตั้งไว้ล่วงหน้ายังคงเป็นอุปกรณ์หลัก
ชนะสำหรับ Ubuntu และชุมชน Linux ที่กว้างขึ้น บริษัท ยังบอกด้วยว่าจะ
ไดรเวอร์อุปกรณ์ต้นน้ำโดยตรงไปยังเคอร์เนล Linux เพื่อช่วยรักษาเสถียรภาพ
และเข้ากันได้ตลอดชีวิตของเวิร์กสเตชันแน่นอนมันง่าย
พอที่จะซื้อแผ่นความคิดที่ใช้ Windows และติดตั้ง Ubuntu ด้วยตนเอง แต่
การสนับสนุนอย่างเป็นทางการหมายถึงผู้ซื้อสามารถมั่นใจได้ว่าฮาร์ดแวร์ทั้งหมดทำงานร่วมกับ
อูบุนตูออกจากกล่องไม่มีไดรเวอร์ที่จะตามล่าหรือกำหนดค่าไฟล์เพื่อแก้ไขอย่างแน่นอน
นอกจากนี้ยังหมายถึงการประหยัดต้นทุนด้วยเช่นกันเนื่องจากไม่จำเป็นต้องมีสิทธิ์ใช้งาน Windows
เลโนโวกล่าวว่าประโยชน์ของ Ubuntu LTS จากวงจรการสนับสนุนห้าปีที่ยาวนาน
การเพิ่มความมั่นใจของผู้ใช้และความเสถียรของระบบในการปรับใช้
เป็นที่ยอมรับ บริษัท ที่อยู่เบื้องหลังประโยชน์ในการดำเนินการรับรองและการถดถอย
การทดสอบระบบเหล่านี้อย่างต่อเนื่องเพื่อให้แน่ใจว่ามันยังคงเป็น
