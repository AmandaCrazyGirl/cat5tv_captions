مەقسىتى لىنۇكىس مېغىزىغا بىر يۈرۈش ياماق يوللىدى
linux ۋە microsoft ئارقىلىق مۇكەممەل مەۋھۇملاشتۇرۇش توپى قۇرۇش
hypervisor ياماقلىرى rfc تەلىپى قىلىپ بېكىتىلگەن
باھا ئۈچۈن ۋە ئەڭ تۆۋەن يولغا قويۇش
مۇلاھىزە ئۈچۈن ئوتتۇرىغا قويۇلغان ھالقىلىق ئۆزگىرىش ياماق بىلەن
kernel linux بولسا hyper-v يىلتىز رايونى سۈپىتىدە ئىجرا بولىدۇ
hyper-v قۇرۇلمىسى يىلتىز رايونىدا بار
قاتتىق دېتالغا بىۋاسىتە ئېرىشىش ۋە بالىلار ئۈچۈن بالىلار رايونى قۇرۇش
مەۋھۇم ماشىنىلاردا ئۇ مىكروسوفتنىڭ پرىنسىپ يۇمشاق دېتال ئىنژېنېرىغا ساھىبخانلىق قىلىدۇ
ۋېي لىيۇ ئۇنى zen نىڭ dom0 غا ئوخشاش ئويلاپ بېقىڭ دەيدۇ
hyper-v نىڭ قۇرۇلمىسى kvm ياكى ئۇنىڭغا قارىغاندا zen غا تېخىمۇ ئوخشايدۇ
esxi بىلەن لېئو ئۇلارنىڭ ئەمەلىيەتتە سىزغانلىقىنى ئېتىراپ قىلدى

windows microsoft نى ئىجرا قىلىشمۇ ئېغىزغا ئۇلانغان
intel نىڭ ئوچۇق مەنبەلىك بۇلۇت نازارەتچىسى نورمال مەۋھۇم ماشىنا نازارەتچىسى
linux يادروسىغا ياسالغان kvm دا ئىجرا بولىدۇ
بۇ ئۆزگىرىشلەر بىلەن microsoft نىڭ ئاز بۇلۇت بولۇشى مۇمكىن
كۆزدە تۇتۇلغان ئۇل ئەسلىھەلەر windows غا قارىغاندا تېخىمۇ كۆپ linux ئىجرا قىلىدۇ
2019-يىلى 7-ئايدا مىكروسوفت ئىنژېنېرى ساشا لېۋىن تەرىپىدىن ئېتىراپ قىلىندى

ئەمما بۇنى مۇكەممەل linux قاتارىغا ئايلاندۇرۇش ئىقتىدارنى ئۆستۈرۈشى مۇمكىن
